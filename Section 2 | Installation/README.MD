# Section 2 | Installation

## Installation

- install node.js from the [official site](https://nodejs.org/en/download)
- verify node.js and npm is installed with the following command `node -v` and `npm -v` respectively 
- Install typescript globally with the following command `npm install -g typescript@5.8` 
- Verify typescript is installed with `tsc -v`

## TypeScript Playground 

- we can write TS code in this [playground](https://www.typescriptlang.org/) too 

## How we use TS 

- TypeScript files use **`.ts`** extension.
- To convert TypeScript into JavaScript, use: `tsc filename.ts`
- TypeScript code finally becomes **normal `.js` file**.
